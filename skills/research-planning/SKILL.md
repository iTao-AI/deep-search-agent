# Research Planning

Create a bounded plan before starting research.

- Break the request into explicit research questions.
- Delegate only to `network_search`, `database_query`, or `knowledge_base`.
- Select sources according to the server-owned profile and runtime context.
- Keep plans and working notes under `/workspace/`.
- Record unavailable sources and evidence gaps instead of inventing facts.
- Do not create or mutate ResearchRun, Evidence, review, verification,
  publication, or delivery records.
